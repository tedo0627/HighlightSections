# HighlightSections
This plugin decorates string with Minecraft Formatting codes.<br>

<img src="https://github.com/tedo0627/HighlightSections/blob/master/image/sample.png?raw=true">

### Support format code
<ul>
    <li>0~9
    <li>a~g
    <li>l~o
    <li>r
</ul>

### Support language
<ul>
    <li>java
    <li>kotlin
    <li>php
</ul>
